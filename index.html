1)Twitter Data Sentiment Analysis on COVID-19 tweets

  • Predict the COVID-19 tweets sentiments and build a model to analyse the tweet belongs to which category of sentiments.
  • Technology used: Twitter API, Python, Logistic Regression Classifier, Streamlit framework
  • Description: Extracting the COVID-19 related tweets using Twitter API and storing as CSV file. Libraries like TextBlob used to find the sentiments, neattext - used to preprocess the data. CountVectorizer, Tf-idf are used to preprocess the text and tokenize the sentence to convert into vector form. Training & prediction is made by building a Logistic Regression Classifier and deployed in heroku platform with the help of Streamlit framework.
